# 📊 E-Commerce Business Intelligence Dashboard

A full end-to-end BI project analyzing e-commerce sales data — from Excel to SQL to Python to a live interactive dashboard.

## 🔗 Live Dashboard
[Click here to view the live dashboard](https://ecommerce-bi-dashboard-fx5fm6ck7rbja6tfvqz73z.streamlit.app)

## 📌 Project Overview
This dashboard helps management analyze sales performance, profitability, and customer behavior to guide business decisions — identifying loss-making categories, top-performing regions, and sales trends.

## 🛠️ Tools & Tech Stack
- **Excel** — initial KPI analysis and pivot tables
- **MySQL** — relational data storage and business queries
- **Python (Pandas, Plotly)** — data cleaning and analysis
- **Streamlit** — interactive web dashboard
- **GitHub + Streamlit Cloud** — deployment

## 📈 Features
- Total Sales, Profit, Orders, Quantity, Customers KPIs
- Category & Sub-Category analysis
- State-wise & City-wise sales breakdown
- Monthly sales trends
- Top customers and top-selling products
- Loss-making category/sub-category identification
- Interactive filters (State, City, Category, Sub-Category, Date range)

## 📁 Dataset
Two related datasets joined on Order ID — order details (amount, profit, quantity, category) and customer/location information (date, customer name, state, city).
